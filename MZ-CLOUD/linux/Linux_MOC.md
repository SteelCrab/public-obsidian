## 빠른 참조

| 상황 | 명령어 |
|------|--------|
| 사용자 추가 | `useradd -m <user>` |
| 비밀번호 설정 | `passwd <user>` |
| 사용자 목록 | `cat /etc/passwd` |
| 그룹 추가 | `groupadd <group>` |
