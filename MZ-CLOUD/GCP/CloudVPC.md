# Cloud VPC 블루프린트

---

GCP 콘솔에서 VPC 생성 시 참고하는 입력값 템플릿

## VPC 네트워크 블루프린트

> 📍 VPC 네트워크 > VPC 네트워크 만들기

| **항목** | **값** | **권장 설정** |
|---------|--------|-------------|
| **이름** | | `vpc-production-kr` |
| **설명** | | `운영 환경 메인 네트워크 (Terraform 관리)` |
| **서브넷 생성 모드** | | **`커스텀`** (AWS처럼 직접 관리하려면 필수) |
| **MTU** | | `1460` (기본값) |
| **IPv6 ULA 설정** | | `사용 안함` |

## 서브넷 설정

| **항목** | **값** | **권장 설정** |
|---------|--------|-------------|
| **이름** | | `subnet-kr-web-01` |
| **리전** | | `asia-northeast3` (서울) |
| **IP 스택 유형** | | `IPv4 (단일 스택)` |
| **IPv4 범위** | | `10.0.1.0/24` |
| **비공개 Google 액세스** | | **`사용`** (Private 서브넷에서 GCS/API 접근 시 필수) |
| **흐름 로그** | | `사용 안함` (비용 절감) |

## 방화벽 및 라우팅

| **항목** | **값** | **권장 설정** |
|---------|--------|-------------|
| **방화벽 규칙** | | `allow-ssh` (22), `allow-icmp` (Ping) 체크 |
| **동적 라우팅 모드** | | **`전역 (Global)`** (리전 간 통신 원활) |
| **DNS 구성** | | `(기본값 유지)` |

## 동적 라우팅 모드

| **모드** | **설명** |
|---------|---------|
| **지역 (Regional)** | 라우터가 있는 리전의 서브넷 경로만 학습하고 알림 |
| **전역 (Global)** | 리전 상관없이 VPC 내 모든 서브넷 경로를 학습하고 알림 (GCP 권장) |
