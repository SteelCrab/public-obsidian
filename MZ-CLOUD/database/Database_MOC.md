## 빠른 참조

| 상황 | 명령어 |
|------|--------|
| DB 접속 | `mysql -u root -p` |
| DB 목록 | `SHOW DATABASES;` |
| 테이블 목록 | `SHOW TABLES;` |
| 테이블 구조 | `DESC <테이블>;` |
| 백업 | `mysqldump -u root -p <db> > backup.sql` |
