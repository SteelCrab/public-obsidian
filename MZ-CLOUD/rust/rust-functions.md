# Rust 함수

#rust #함수 #fn

---

`fn` 키워드로 함수를 정의한다. snake_case 네이밍을 사용한다.

## 기본 구조

```rust
fn add(x: i32, y: i32) -> i32 {
    x + y    // 마지막 표현식이 반환값 (세미콜론 없음)
}

fn print_value(x: i32) {
    println!("값: {}", x);  // 반환값 없음 (unit 타입)
}
```

## 매개변수와 반환

| 문법 | 설명 |
|------|------|
| `fn foo(x: i32)` | 매개변수 (타입 필수) |
| `-> i32` | 반환 타입 지정 |
| `x + y` | 마지막 표현식 = 반환값 |
| `return x + y;` | 명시적 반환 (조기 반환 시) |
| `-> ()` | unit 타입 (반환값 없음, 생략 가능) |

## 표현식 vs 구문

```rust
let y = {
    let x = 3;
    x + 1          // 표현식 - 값을 반환 (세미콜론 없음)
};                 // y = 4

let z = {
    let x = 3;
    x + 1;         // 구문 - 세미콜론이 있으면 ()를 반환
};                 // z = ()
```

> [!warning] 세미콜론 주의
> 함수 본문의 마지막 표현식에 세미콜론을 붙이면 `()`가 반환된다.
