# Rust 변수와 가변성

#rust #변수 #가변성

---

Rust의 변수는 기본적으로 불변(immutable)이다.

## 변수 선언

```rust
let x = 5;           // 불변 변수
let mut y = 10;      // 가변 변수
y = 20;              // OK

const MAX: u32 = 100_000;  // 상수 (타입 필수)
static HELLO: &str = "hello"; // 정적 변수
```

## 불변 vs 가변

| 키워드 | 설명 |
|--------|------|
| `let x = 5` | 불변 - 재할당 불가 |
| `let mut x = 5` | 가변 - 재할당 가능 |
| `const X: i32 = 5` | 상수 - 컴파일 타임에 결정 |
| `static X: i32 = 5` | 정적 - 프로그램 수명 동안 존재 |

## 섀도잉 (Shadowing)

같은 이름으로 새 변수를 선언하여 이전 변수를 가린다.

```rust
let x = 5;
let x = x + 1;       // 섀도잉 (타입 변경 가능)
let x = "hello";     // 타입이 달라도 OK
```

> [!info] 섀도잉 vs mut
> 섀도잉은 새 변수를 만드는 것이고, `mut`는 같은 변수를 변경하는 것이다.
> 섀도잉은 타입 변경이 가능하지만 `mut`는 불가능하다.
