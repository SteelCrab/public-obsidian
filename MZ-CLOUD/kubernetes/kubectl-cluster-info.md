# Kubectl 클러스터 정보

#kubernetes #kubectl #클러스터

---

클러스터 상태 및 정보 확인.

| 명령어 | 설명 |
|--------|------|
| `kubectl cluster-info` | 클러스터 정보 |
| `kubectl get nodes` | 노드 목록 |
| `kubectl get nodes -o wide` | 노드 상세 목록 |
| `kubectl describe node <노드>` | 노드 상세 정보 |
| `kubectl version` | 버전 확인 |
| `kubectl api-resources` | API 리소스 목록 |
