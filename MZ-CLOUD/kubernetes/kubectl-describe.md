# Kubectl 리소스 상세

#kubernetes #kubectl #describe

---

리소스 상세 정보 확인.

| 명령어 | 설명 |
|--------|------|
| `kubectl describe pod <이름>` | Pod 상세 |
| `kubectl describe svc <이름>` | 서비스 상세 |
| `kubectl describe deploy <이름>` | 디플로이먼트 상세 |
| `kubectl describe node <이름>` | 노드 상세 |
| `kubectl describe ingress <이름>` | 인그레스 상세 |
| `kubectl describe pvc <이름>` | PVC 상세 |
