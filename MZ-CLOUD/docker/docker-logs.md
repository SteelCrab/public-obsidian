# Docker 로그와 모니터링

#docker #logs #모니터링

---

컨테이너 로그 및 상태 확인.

| 명령어 | 설명 |
|--------|------|
| `docker logs <컨테이너>` | 로그 확인 |
| `docker logs -f <컨테이너>` | 로그 실시간 확인 |
| `docker logs --tail 100 <컨테이너>` | 최근 100줄 |
| `docker stats` | 리소스 사용량 |
| `docker top <컨테이너>` | 프로세스 목록 |
| `docker inspect <컨테이너>` | 상세 정보 |
