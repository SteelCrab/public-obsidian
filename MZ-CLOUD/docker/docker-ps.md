# Docker 컨테이너 관리

#docker #ps #컨테이너

---

컨테이너 관리 명령어.

| 명령어 | 설명 |
|--------|------|
| `docker ps` | 실행 중인 컨테이너 |
| `docker ps -a` | 모든 컨테이너 |
| `docker start <컨테이너>` | 컨테이너 시작 |
| `docker stop <컨테이너>` | 컨테이너 중지 |
| `docker restart <컨테이너>` | 컨테이너 재시작 |
| `docker rm <컨테이너>` | 컨테이너 삭제 |
| `docker rm -f <컨테이너>` | 강제 삭제 |
| `docker container prune` | 중지된 컨테이너 정리 |
