# Docker 네트워크

#docker #network #네트워크

---

컨테이너 네트워크 관리.

| 명령어 | 설명 |
|--------|------|
| `docker network ls` | 네트워크 목록 |
| `docker network create <이름>` | 네트워크 생성 |
| `docker network rm <이름>` | 네트워크 삭제 |
| `docker network connect <네트워크> <컨테이너>` | 네트워크 연결 |
| `docker network disconnect <네트워크> <컨테이너>` | 네트워크 분리 |
| `docker network inspect <이름>` | 네트워크 정보 |
