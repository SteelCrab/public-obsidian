# Docker 이미지

#docker #이미지

---

이미지 관리 명령어.

| 명령어                      | 설명         |
| ------------------------ | ---------- |
| `docker images`          | 이미지 목록     |
| `docker pull <이미지>`      | 이미지 다운로드   |
| `docker build -t <이름> .` | 이미지 빌드     |
| `docker rmi <이미지>`       | 이미지 삭제     |
| `docker image prune`     | 미사용 이미지 정리 |
| `docker tag <이미지> <새이름>` | 이미지 태그     |
| `docker push <이미지>`      | 레지스트리에 푸시  |
