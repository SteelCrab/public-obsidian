# Git Blame

#git #blame #추적

---

파일의 각 라인을 누가 마지막으로 수정했는지 확인한다.

| 명령어 | 설명 |
|--------|------|
| `git blame <파일>` | 전체 파일 blame |
| `git blame -L 10,20 <파일>` | 10-20번 라인만 |
| `git blame -L 10,+5 <파일>` | 10번부터 5줄 |
| `git blame -e <파일>` | 이메일 표시 |
| `git blame -w <파일>` | 공백 변경 무시 |
| `git blame -M <파일>` | 파일 내 이동 감지 |
| `git blame -C <파일>` | 파일 간 복사 감지 |

## 출력 예시

```
abc1234 (홍길동 2024-01-15 10:30) function login() {
def5678 (김철수 2024-01-20 14:22)   const user = getUser();
```

> [!tip] 활용
> - 버그 원인 추적
> - 코드 변경 이유 파악 (커밋 메시지 확인)
> - 담당자 찾기
