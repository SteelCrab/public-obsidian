# Git 스테이징 (add)

#git #add #스테이징

---

변경사항을 스테이징 영역에 추가한다.

| 명령어 | 설명 |
|--------|------|
| `git add <파일>` | 특정 파일 스테이징 |
| `git add <파일1> <파일2>` | 여러 파일 |
| `git add .` | 현재 디렉토리 전체 |
| `git add -A` | 모든 변경사항 |
| `git add -u` | 수정/삭제만 (새 파일 제외) |
| `git add -p` | 대화형 (hunk 단위) |
| `git add *.js` | 패턴으로 스테이징 |

## 대화형 스테이징 (-p)

파일의 일부분만 선택적으로 스테이징할 수 있다.

| 키 | 설명 |
|----|------|
| `y` | 이 hunk 스테이징 |
| `n` | 건너뛰기 |
| `s` | hunk를 더 작게 분할 |
| `q` | 종료 |
