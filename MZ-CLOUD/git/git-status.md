# Git 상태 확인

#git #status #상태

---

작업 디렉토리와 스테이징 영역의 상태를 확인한다.

| 명령어 | 설명 |
|--------|------|
| `git status` | 전체 상태 |
| `git status -s` | 간략한 상태 (short) |
| `git status -sb` | 간략 + 브랜치 정보 |
| `git status --ignored` | 무시된 파일도 표시 |

## 상태 기호 (-s 옵션)

| 기호 | 의미 |
|------|------|
| `M` | 수정됨 (Modified) |
| `A` | 추가됨 (Added) |
| `D` | 삭제됨 (Deleted) |
| `R` | 이름변경 (Renamed) |
| `??` | 추적되지 않음 |
| `!!` | 무시됨 |

> [!info] 두 열의 의미
> 첫 번째 열: 스테이징 영역 상태
> 두 번째 열: 작업 디렉토리 상태
