# Git 사용자 설정

#git #config #설정

---

커밋에 기록될 사용자 정보를 설정한다.

| 명령어 | 설명 |
|--------|------|
| `git config --global user.name "이름"` | 전역 사용자 이름 |
| `git config --global user.email "이메일"` | 전역 이메일 |
| `git config user.name "이름"` | 현재 저장소 사용자 이름 |
| `git config user.email "이메일"` | 현재 저장소 이메일 |

> [!tip] 회사/개인 프로젝트 분리
> 전역 설정은 개인용, 각 저장소에서 회사 이메일 설정 가능
