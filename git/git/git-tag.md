# Git Tag

#git #tag #태그 #버전

---

특정 커밋에 이름표를 붙인다. 주로 릴리스 버전 관리에 사용.

## 조회

| 명령어 | 설명 |
|--------|------|
| `git tag` | 태그 목록 |
| `git tag -l "v1.*"` | 패턴 검색 |
| `git show <태그>` | 태그 상세 |

## 생성

| 명령어 | 설명 |
|--------|------|
| `git tag <태그명>` | 경량 태그 |
| `git tag -a <태그명> -m "메시지"` | 주석 태그 |
| `git tag -a <태그명> <커밋>` | 특정 커밋에 |

## 삭제/공유

| 명령어 | 설명 |
|--------|------|
| `git tag -d <태그명>` | 로컬 삭제 |
| `git push origin <태그명>` | 태그 푸시 |
| `git push origin --tags` | 모든 태그 푸시 |
| `git push origin --delete <태그명>` | 원격 삭제 |

## 경량 vs 주석 태그

| | 경량 | 주석 |
|-|------|------|
| 메타데이터 | 없음 | 작성자, 날짜, 메시지 |
| 용도 | 임시 표시 | 릴리스 (권장) |
