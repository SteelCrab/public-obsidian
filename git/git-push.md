# Git Push

#git #push #원격

---

로컬 커밋을 원격 저장소에 업로드한다.

| 명령어 | 설명 |
|--------|------|
| `git push` | 현재 브랜치 푸시 |
| `git push origin <브랜치>` | 특정 브랜치 푸시 |
| `git push -u origin <브랜치>` | 업스트림 설정 후 푸시 |
| `git push --all` | 모든 브랜치 푸시 |
| `git push --tags` | 모든 태그 푸시 |
| `git push origin --delete <브랜치>` | 원격 브랜치 삭제 |
